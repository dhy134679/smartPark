# 基于微信小程序的智能停车场系统 —— 技术定位文档

## 一、项目概述

### 1.1 项目背景

随着城市化进程加速，老旧小区普遍面临车位数量不足、停车资源紧张的问题。传统的停车管理方式效率低下，缺乏智能化手段，导致车位利用率低、外来车辆管理混乱、车主停车体验差。

### 1.2 项目目标

设计并实现一个**基于微信小程序的智能停车场管理系统**，利用深度学习、计算机视觉等人工智能技术，实现车牌自动识别、车位智能管理、停车费用自动计算、空闲车位预测等功能，提升老旧小区的停车管理效率和车主停车体验。

### 1.3 系统核心功能

| 序号 | 功能模块 | 功能描述 |
|------|---------|---------|
| 1 | 车位管理 | 小区车主可上传车位空闲信息，系统实时展示空闲车位 |
| 2 | 车牌识别 | 基于 PyTorch 深度学习模型，自动识别车辆车牌号码 |
| 3 | 车辆判别 | 自动判断进入车辆是否为小区注册车辆 |
| 4 | 计时收费 | 外来车辆按停靠时间自动计费，支持模拟支付 |
| 5 | 车位导航 | 停车场内部路线规划，引导车辆至空闲车位 |
| 6 | 车位预测 | 车位已满时，基于深度学习预测下一个空闲车位的出现时间 |
| 7 | 出入模拟 | 模拟车辆出入场景，演示完整的识别→判断→计费流程 |

## 四、核心算法说明

### 4.1 车牌识别算法

采用 **两阶段** 车牌识别方案，全部基于 PyTorch 实现，不依赖任何外部 OCR 服务：

```
输入图像 → YOLOv5 检测车牌区域 → 裁剪车牌图像 → LPRNet 识别字符 → 输出车牌号
```

**第一阶段 —— 车牌检测（YOLOv5）**
- 采用 YOLOv5s 轻量级模型，输入分辨率 640×640
- 使用 CCPD（Chinese City Parking Dataset）中国城市停车数据集进行训练
- 输出车牌区域的边界框坐标（Bounding Box）

**第二阶段 —— 字符识别（LPRNet）**
- LPRNet（License Plate Recognition Network）是一种轻量级端到端识别网络
- 网络结构：Backbone（特征提取）→ Squeeze → 全连接层 → CTC 解码
- 支持识别蓝牌、绿牌（新能源）等多种中国车牌类型
- 字符集包含：省份简称（京津沪渝等）、字母（A-Z）、数字（0-9）

**训练数据集**
- CCPD 2019/2020 数据集：约 30 万张中国车牌图像
- 包含多角度、多光照、多天气条件下的车牌样本

### 4.2 车位预测算法

采用 **LSTM（长短期记忆网络）** 进行车位占用率的时序预测：

```
历史车位数据（时间序列）→ 数据预处理 → LSTM 模型 → 预测未来空闲车位时间
```

**模型结构**
- 输入层：过去 N 个时间步的车位占用数据（occupancy rate, 时间特征）
- LSTM 层：2 层 LSTM，隐藏层维度 128
- 全连接层：映射到预测输出
- 输出层：未来 M 个时间步的车位占用率预测

**特征工程**
- 时间特征：小时、星期几、是否工作日、是否节假日
- 历史占用率：过去 24 小时的车位占用率序列
- 统计特征：历史同时段平均占用率

**数据来源**
- 系统运行过程中自动采集车位变化数据
- 初始阶段使用模拟生成的历史数据进行预训练

### 4.3 停车场内部导航算法

采用 **A\* 寻路算法** 进行停车场内部路径规划：

```
车辆入口位置 → A* 路径搜索 → 最优行驶路线 → 到达目标空闲车位
```

- 将停车场平面图网格化，建立可通行区域和障碍物地图
- 以入口为起点，空闲车位为终点，计算最短路径
- 使用曼哈顿距离作为启发函数
- 在小程序端通过 Canvas 绘制导航路线

## 九、技术难点与解决方案

| 难点 | 解决方案 |
|------|---------|
| 车牌识别精度 | 采用 CCPD 大规模数据集训练，YOLOv5 检测 + LPRNet 识别两阶段方案提高精度 |
| 车牌识别不依赖外部服务 | 全部使用 PyTorch 本地推理，模型权重文件随项目部署 |
| 车位预测准确性 | LSTM 模型结合时间特征（小时/星期/节假日），初始数据不足时使用模拟数据预训练 |
| 停车场内部导航 | 自研 A* 寻路算法，将停车场抽象为网格地图，Canvas 绘制导航路线 |
| 实时车位状态同步 | WebSocket 实时推送车位变化，保证多端数据一致 |
| 本地运行无需部署 | MySQL 本地安装运行，所有服务 localhost 运行 |

## 十、技术创新点

1. **端到端车牌识别系统**：基于 PyTorch 的 YOLOv5 + LPRNet 两阶段车牌识别，完全自主实现，不依赖任何第三方 OCR 服务。
2. **LSTM 深度学习车位预测**：利用长短期记忆网络对历史停车数据进行时序分析，智能预测车位空闲时间。
3. **停车场内部智能导航**：基于 A* 算法的室内路径规划，结合 Canvas 动态渲染导航路线。
4. **完整的出入模拟演示系统**：提供可视化的车辆出入场模拟，完整演示从车牌识别到计费收费的全流程。
