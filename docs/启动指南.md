# 智能停车场系统 —— 启动指南

## 环境要求

- Python 3.10+
- MySQL 8.0+
- Node.js 18+（UniApp 编译用）
- 微信开发者工具
- HBuilderX（推荐，用于前端开发）

## 一、数据库准备

### 1. 启动 MySQL 服务

确保本地 MySQL 已安装并运行。推荐使用 phpStudy / XAMPP 一键启动。

### 2. 创建数据库并初始化表

打开 MySQL 客户端（Navicat / 命令行 / phpMyAdmin），执行初始化脚本：

```bash
mysql -u root -p < backend/sql/init.sql
```

或者在 Navicat 中直接打开 `backend/sql/init.sql` 并执行。

## 二、后端启动

### 1. 安装 Python 依赖

```bash
cd d:\p1\backend
pip install -r requirements.txt
```

### 2. 配置数据库连接

编辑 `backend/.env` 文件，修改数据库密码：

```env
DATABASE_URL=mysql+aiomysql://root:你的密码@127.0.0.1:3306/smart_parking?charset=utf8mb4
```

### 3. 初始化 ORM 并填充模拟数据

```bash
cd d:\p1\backend
python init_db.py
```

这会自动创建 ORM 表并插入 7 天的模拟车位状态数据（用于 LSTM 预测）。

### 4. 启动后端服务

```bash
cd d:\p1\backend
python -m uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

启动后访问 http://localhost:8000/docs 查看 API 文档。

## 三、前端启动

### 方式一：HBuilderX（推荐）

1. 用 HBuilderX 打开 `d:\p1\frontend` 目录
2. 点击 **运行 → 运行到小程序模拟器 → 微信开发者工具**
3. 自动编译并打开微信开发者工具

### 方式二：命令行

```bash
cd d:\p1\frontend
npm install
npm run dev:mp-weixin
```

编译完成后，用微信开发者工具打开 `frontend/dist/dev/mp-weixin` 目录。
