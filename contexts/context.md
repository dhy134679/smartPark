# 智能停车场系统 - 项目上下文

## 项目信息
- **名称**: 基于微信小程序的智能停车场系统
- **类型**: 毕业设计
- **运行环境**: Windows localhost
- **数据库**: MySQL 8.0 (smart_parking)

## 技术栈
- **前端**: UniApp (Vue3) → 微信小程序
- **后端**: Python FastAPI + SQLAlchemy 2.0
- **AI**: PyTorch (YOLOv5+LPRNet 车牌识别, LSTM 车位预测)
- **认证**: JWT
- **导航**: A* 寻路算法 + Canvas

## 当前进度
- ✅ 全部设计文档已完成（docs/ 目录）
- ✅ 数据库初始化脚本已完成（backend/sql/init.sql）
- ⬜ 后端代码待开发
- ⬜ 前端代码待开发（UniApp Vue3 项目已初始化，在 frontend/ 目录）

## 关键设计决策
1. 数据库用 MySQL 而非 SQLite（毕设答辩更专业）
2. 车牌识别用预训练模型，不从零训练
3. 不接外部地图 API，停车场内部用 A* 自研导航
4. 不用 Redis/消息队列等中间件（localhost 场景不需要）
5. 模拟支付，不接入真实微信支付
